# A dream youve had before Nightmare solver

A tool to solve the codes for the nightmare puzzle.

> [!NOTE]
> These notes were transcribed manually. Some entries may contain mistakes due to ambiguous spacing or characters that can't be reliably transcribed. Notes with emojis are especially difficult to transcribe so keep an eye on them

Feel free to open a PR if you find an incorrect transcription. Preferably with a screenshot of the codes and final 5 letter code included.
